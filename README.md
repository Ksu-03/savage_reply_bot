# ОТВЕЧАЛКА — Telegram бот

Бот выдаёт 3 варианта ответа на любую жизненную ситуацию.  
Дерзко, но не грязно. Для аудитории 15–25 лет.

## Функции
- /start — приветствие и главное меню
- /ask — описать ситуацию → получить 3 ответа
- /top — топ лучших ответов по версии пользователей
- /premium — платные плюшки

## Технологии
- Python + aiogram 3.x
- Хранение — JSON / SQLite
- Деплой — любой хостинг
